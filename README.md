# ClarityPay Take-Home: Consumer Credit Analytics

**Question**: *"Are we lending to the right borrowers?"*
**Dataset**: Lending Club 2007–2020 Q3 (Kaggle)

---

## Project Structure

```
ClarityPay/
├── analysis.ipynb          # Main Jupyter Notebook (all analysis)
├── archive/
│   ├── Loan_status_2007-2020Q3.gzip   # Raw data (~1.7 GB compressed)
│   └── LCDataDictionary.xlsx          # Field definitions (151 columns)
├── figs/                   # All charts generated by the notebook
└── README.md               # This file
```

## How to Run

**Requirements**: Python 3.10+, and the following packages:

```
pandas >= 1.5
numpy
matplotlib
seaborn
openpyxl        # for reading .xlsx data dictionary
jupyter
```

Install with:

```bash
pip install pandas numpy matplotlib seaborn openpyxl jupyter
```

**Run the notebook**:

```bash
jupyter notebook analysis.ipynb
```

Then run all cells top-to-bottom (`Cell → Run All`).

> The data loading step (Section 2) processes the 1.7 GB file in chunks and takes approximately 3–5 minutes depending on hardware. All subsequent cells run in seconds.

---

## What the Notebook Does

| Section | Content |
|---|---|
| 1 · Setup | Imports, paths, column list |
| 2 · Data Loading | Chunked read of gzip CSV; filter to completed loans only |
| 3 · Cleaning & Target | Binary default flag; clean string columns; add derived fields |
| 4 · EDA | Outcome distribution; volume/bad rate by year; feature distributions |
| 5 · Risk Analysis | Default rate by FICO, grade, revolving util, DTI, derogatory marks, purpose, term |
| 6 · Policy Rules | Six evidence-based decline rules with business rationale |
| 7 · Trade-off Analysis | Combined rulesets tested; trade-off scatter plot |
| 8 · Summary | Key findings and recommended policy |

## Key Findings (Summary)

- **Overall bad rate**: ~19.5% of completed loans charged off
- **Top 3 risk drivers**: FICO score, Lending Club loan grade, revolving credit utilization
- **Recommended policy** (FICO < 680 OR Grade E/F/G OR Revol Util > 80%):
  - Bad rate reduction: ~21% relative
  - Volume declined: ~43%
  - Good loans rejected: ~41%
